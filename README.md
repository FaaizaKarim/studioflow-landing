# Studioflow  SaaS Landing Page

Conversion-focused landing page for a fictional project-management SaaS for creative studios. Portfolio project by Faaiza Saand.

Hero, feature grid, 3-tier pricing, testimonials, FAQ accordion, email-capture CTA. Responsive mobile-first layout, semantic HTML, SEO/OG metadata, scroll-reveal animations.

## Stack

Next.js 14 (App Router), React 18, Tailwind CSS 3.

## Run locally

```bash
npm install
npm run dev
```

Open http://localhost:3000

## Deploy

Push to GitHub and import into Vercel. No configuration needed.
